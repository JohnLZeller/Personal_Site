Job Scheduler API for Firefox OS

September 2014

For my senior project I am aiming to build a job scheduler API for Firefox OS. Firefox OS phones have launched in countries around the world, many of them with a poorer quality of infrastructure than what you would find in a more developed country. For that reason, network reliability can become a huge problem, saping battery life, costing more, and overwhelming what infrastructure does exist. This project is to create a task scheduler API that allows the phone to delay a task until it is on a good network. Doing this improves efficiency and user experience by decreasing unnecessary timeouts. Read More
